Understanding Traefik

When using Docker as a provider, “Services” are the docker-compose service definitions.

Regarding ports: If a container exposes multiple ports, or does not expose any port, then you must manually specify which port Traefik should use for communication by using the label traefik.http.services.<service_name>.loadbalancer.server.port (Read more on this label in the dedicated section in routing) source
